Privacy Policy for FocusScroll

Effective Date: March 22, 2026

1. Information We Collect

FocusScroll does not collect, store, or share any personal information. The app does not require user accounts, and no data is sent to external servers except for the YouTube API calls necessary to fetch video information.

2. YouTube API Usage

This app uses the YouTube Data API to display educational videos based on user searches. The API key is used only for fetching video metadata and embeds. We do not access any user-specific YouTube data.

3. Data Storage

Your preferences (such as sound setting) are stored locally on your device using browser storage. No data is transmitted to us.

4. Third-Party Services

YouTube videos are embedded via YouTube's iframe player. YouTube's own privacy policy applies to the video content. You can view YouTube's privacy policy at: https://policies.google.com/privacy.

5. Children's Privacy

FocusScroll is intended for general audiences and does not knowingly collect any personal information from children under 13.
